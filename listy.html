Listy

li
Jest to znacznik służący do wyświetlania elementów listy.

Kod

        
        1 <ol>
        2    <li>
        3       Pierwszy element listy
        4    </li>
        5    <li>
        6       Drugi element listy
        7    </li>
        8 </ol>

Domyślny styl

        
        li{
            display: list-item;
        }
ol
Tworzy listę uporządkowaną, czyli ponumerowaną. Numerom można nadać różny styl, aby były wyświetlane np. alfabetycznie.

    list-style-type:

  1. armenian
  2. cjk-ideographic
  3. decimal
  4. decimal-leading-zero
  5. georgian
  6. hebrew
  7. hiragana
  8. hiragana-iroha
  9. katakana
  10. katakana-iroha
  11. lower-alpha
  12. lower-greek
  13. lower-latin
  14. lower-roman
  15. upper-alpha
  16. upper-latin
  17. upper-roman
  18. none
  19. inherit

Kod

        
        1 <ol>
        2    <li>
        3       Pierwszy element listy
        4    </li>
        5    <li>
        6       Drugi element listy
        7    </li>
        8 </ol>

Domyślny styl

        
        ol{
            display: block;
            list-style-type: decimal;
            margin-top: 1em;
            margin-bottom: 1em;
            margin-left: 0;
            margin-right: 0;
            padding-left: 40px;
        }
ul
Tworzy listę nieuporządkowaną, czyli wypunktowaną. Żeby nie było widać punktów można użyć list-style-type: none;

Kod

        
        1 <ul>
        2    <li>
        3       Pierwszy element listy
        4    </li>
        5    <li>
        6       Drugi element listy
        7    </li>
        8 </ul>

Domyślny styl

        
        ul{
            display: block;
            list-style-type: disc;
            margin-top: 1em;
            margin-bottom: 1 em;
            margin-left: 0;
            margin-right: 0;
            padding-left: 40px;
        }
