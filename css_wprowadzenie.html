Css
To arkusz stylów. Ważne jest zachowanie kaskadowości stylów. Kaskadowość określa, który styl(w przypadku dołączenia kilku) zostanie wyświetlony przez przeglądarkę. Gdy arkusz zawnętrzny dołącza się do pliku html trzeba wpisać w head <link href="nazwa_pliku.css" rel="stylesheet">, a gdy używa się wewnętrznego arkusza stylów, należy to robić wewnątrz tagu <style>.

Kaskadowość:

  1. Styl lokalny
  2. Rozciąganie stylu(<span>)
  3. Wydzielone bloki div
  4. Wewnętrzny arkusz stylu
  5. Zewnętrzny arkusz
  6. Import arkusza stylów
  7. Domyślne ustawienia przeglądarki
